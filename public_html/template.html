Navbar

To add the page to the navbar, you will need to edit ./js/script.js. In the addNav function, there is a constant variable called pages. After the last page, add a new line and add in the information for the new page in a similar fashion as before. The key is the text that will appear in the navigation bar, such as Template for this page. If the page does not exist, you can set the value to null or [null, iconValue], where the iconValue is the FontAwesome icon that will appear when the mouse hovers over the button. If you don't want to specify an icon, make sure you set the value as null.
